Intro-to-python

This contains files and documentation for the intro to python workshop put on by the physics society at UOIT.

Download this project as a .zip file Download this project as a tar.gz file

Overview

This is an intro to using python. We will go over basic skills and make a program to solve projectile motion problems. You should have Linux installed on your laptop. If you do not and would like to come to this workshop please email info@uoitphysics.ca.

Tutorial

We will build all of the python scripts from scratch. If you would like to examine the finished products you can download them using the .tar.gz button above. To extract the files run the command 

$ tar -zxvf uoit-research-workshops-intro-to-python-ee046b6.tar.gz

You can run python programs in the terminal one of two ways. Use 

$ python my_python_script.py

or

$ chmod +x my_python_script.py
$ ./my_python_script.py

If you would like to use the interpreter directly (which is useful to test things quickly) just type 

$ python

into the terminal.
